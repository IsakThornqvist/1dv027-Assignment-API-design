## Dependencies

- `@apollo/server` — runs the GraphQL server
- `@prisma/client` — database client for PostgreSQL
- `bcrypt` — hashes passwords
- `cors` — allows cross-origin requests
- `csv-parse` — parses the CSV seed file
- `dotenv` — loads .env variables
- `express` — web server framework
- `graphql` — GraphQL language support
- `jsonwebtoken` — creates and verifies JWT tokens

## devDependencies

- `nodemon` - auto restarts server when you save a file
- `prisma` - CLI tool for migrating, seeding, studio
